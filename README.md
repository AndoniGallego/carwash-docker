# carwash-docker
Docker metadata for carwash

# Setup API

It's neccesary clone the API repository in the folder api

```bash
$ cd api
$ git clone git@github.com:AndoniGallego/carwash-api.git
```

# Build with Docker

```bash
$ docker compose build
```

# Run with Docker

```bash
$ docker compose up
```

# Run with Docker in background

```bash
$ docker compose up -d
```
